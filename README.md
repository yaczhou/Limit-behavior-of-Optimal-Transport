# Limit-behavior-of-Optimal-Transport
Studied the asymptotic dynamics of iterated optimal transport maps in Wasserstein space using tools from optimal transport theory and Otto calculus.
In one dimension, characterized limiting fixed points through intersections of cumulative distribution functions and analyzed their stability.
In higher dimensions, investigated stability properties of transport fixed points via the Monge–Ampère equation and verified the theory through Sinkhorn-based numerical simulations on bounded domains.
